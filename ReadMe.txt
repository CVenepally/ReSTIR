STARSHIP


- Launch Starship_Release_x64.exe in Run folder to play the game

How To Play:

- Player has to face 5 waves with each wave having increasing number of enemies
- To clear a wave the player should kill all the beetles and wasps that swarm up to kill the player
- Avoid or destroy asteroids for a better chance to win
- Player has 4 lives throughout the entire game

- Controls

    Key Board Controls
      - W accelerates the ship in the direction you are facing
      - A, D changes ship's direction
      - Space Bar shoots Bullets
      - Q to switch between full auto or single fire
      - E to shoot missiles (after wave 3)
      - N to respawn player
      - ESC to exit to menu/exit to windows

    Controller Controls
       - Left stick to move and aim
       - A to shoot
       - START to Respawn
       - BACK to exit to menu/exit to windows 
- Dev Cheats
  - I to spawn more asteroids
  - F1 to use debug mode
  - F8 to hard reset the game
  - P to pause the game
  - T to slow down time
  - O to pause the game after playing a frame
